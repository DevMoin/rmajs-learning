jab tak condition true hogi body ka code run hoga.
steps   condition;body;condition;body;condition........
structure
```
while(condition)
body

while(condition)
{
    body
}
```

do-While loop:
jab tak condition true hogi body ka code run hoga.
steps   body;condition........

```
do{
    body
}while(condition)

```
